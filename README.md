# MINESWEEPER

CLI 지뢰찾기!<br>
전에 만들었던 [miner](https://github.com/hooss-only/miner/)과 다르게 ncurses.h를 활용하여 더 편하게 즐길 수 있습니다.

# How to play

```
git clone https://github.com/hooss-only/minesweeper.git --depth=1
cd minesweeper
mkdir -p ./build/output
make
./build/output/minesweeper
```
